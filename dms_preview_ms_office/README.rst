This module provides a preview of Microsoft Office documents in DMS
file. It allows users to view documents such as Word, Excel, and
PowerPoint files directly within the Odoo interface without needing to
download them.

**Table of contents**

.. contents::
   :local:

Usage
=====

1. Install the module in your Odoo instance.
2. The MS Office file will be previewed when you click on image of the
   file in the kanban view
